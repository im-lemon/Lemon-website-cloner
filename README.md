# 🍋 | LEMON WEBSITE CLONER

Lemon website cloner is a website cloner that uses python's Requests and BeautifulSoup4 packages to fetch HTML, CSS, and images from a website of your choice. It uses argparse for CLI flags, these are required.

# INSTALLATION:

```bash
pip install bs4 requests
```

OR:

```bash
pip install -r requirements.txt
```


# Usage:

``python main.py -i example.com``
The script will then organize the Images, HTML, and CSS files into neat little folders!

That's right; you do not need to enter in "https://www." or "http://www.", the script does that for you!
